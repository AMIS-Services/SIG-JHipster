# SIG JHipster

This repo contains files for the JHipster workshop on 5th of july 2018 

[JHipster](https://www.jhipster.tech) is a development platform to generate, develop and deploy Spring Boot + Angular/React Web applications and Spring microservices.

## Installing JHipster

Setup your environment by following one of the installation types on https://www.jhipster.tech/installation/
